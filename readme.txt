How to run the project: 
			- Github link-> https://github.com/ebullientcreed/CryptoScholar
			- Run komodo nodes on the host machine.
			./komodod -ac_name=TeamCR -ac_supply=200000 -addnode=192.168.43.1:8129 -gen-genproclimit=$(nproc)&
			-Run on virtual box ubuntu
			./komodod -ac_name=TeamCR -ac_supply=200000 -addnode=192.168.43.78:8129 &
			-Run the spring boot application (Eclipse IDE)
			-Navigate to the crypto-scholar folder and do the following:
			-Run the angular application using "ng serve --proxy-config proxy.conf.json" command
